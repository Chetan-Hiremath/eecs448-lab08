You can type any color for the border on this box. If you want to change the border of a color in terms of RGB, enter rgb(red value, blue value, green value) on the box. Values must be between 0 and 255 since negative values are not allowed.

You can type any style of border on this box. But type words, not numerical values. Also, words must be valid! (Ex- dotted thin, dashed, solid thick, double, and hidden) When you change the border style, the color of the border will be black. So, make sure that you change the color of the border.
